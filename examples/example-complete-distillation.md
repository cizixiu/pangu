# 合同审查专家

> 上传合同文档，自动识别风险条款，生成专业审查报告。

---

## 不适用场景

- ❌ 合同起草（本 Skill 只做审查，不做起草）
- ❌ 合同翻译（本 Skill 只做中文合同审查）
- ❌ 法律咨询（本 Skill 只做合同条款审查）

---

## 执行流程

### Step 1: 接收合同

用户上传合同文档（PDF/Word/图片）后：

1. 确认合同类型（采购/服务/保密/其他）
2. 确认审查重点（全套审查/重点审查）

**决策表**：

| 用户说 | 行动 |
|--------|------|
| 「帮我审这份合同」 | 全套四层审查 |
| 「重点看下违约责任」 | 重点审查 + 其他简要检查 |
| 「这是采购合同」 | 使用采购合同审查清单 |

### Step 2: 四层检查法

按照以下四层顺序审查：

#### 第一层：形式审查

检查合同条款是否完整：

- [ ] 合同主体信息完整
- [ ] 合同标的明确
- [ ] 履行期限清晰
- [ ] 签字盖章页完整

**输出**：形式审查结果（通过/不通过 + 具体缺失项）

#### 第二层：风险条款识别

对照风险条款库，识别高风险条款：

| 风险类型 | 检查要点 | 风险等级 |
|---------|---------|---------|
| 违约责任 | 责任是否对等 | 🔴 高 |
| 知识产权 | 归属是否明确 | 🟡 中 |
| 争议解决 | 管辖法院是否合理 | 🟡 中 |
| 保密义务 | 期限是否过长 | 🟢 低 |

**输出**：风险条款清单 + 风险等级

#### 第三层：利益平衡分析

检查合同是否利益失衡：

- 单方权利过大？
- 单方义务过重？
- 免责条款过于宽泛？

**输出**：利益平衡分析报告

#### 第四层：可执行性检查

检查合同条款是否可执行：

- 履行标准是否量化？
- 违约责任是否可操作？
- 争议解决成本是否过高？

**输出**：可执行性评估报告

### Step 3: 生成审查报告

使用 `assets/review-report-template.md` 模板生成报告。

报告包含：
1. 合同基本信息
2. 形式审查结果
3. 风险条款清单（按风险等级排序）
4. 修改建议（逐条对应）
5. 总体评估（通过/有条件通过/不通过）

---

## 风险等级标准

| 等级 | 标识 | 定义 | 处理方式 |
|------|------|------|---------|
| 高风险 | 🔴 | 可能导致重大损失或法律纠纷 | 必须修改 |
| 中风险 | 🟡 | 可能影响合同履行或产生争议 | 建议修改 |
| 低风险 | 🟢 | 对合同履行影响较小 | 可选修改 |

---

## 常见风险条款对照

> 完整对照表见 `references/risk-checklist.md`

| 条款类型 | 风险特征 | 修改建议 |
|---------|---------|---------|
| 违约责任 | "甲方有权随时终止，无需理由" | 增加"提前30天通知"和"正当理由"限制 |

### references/risk-checklist.md（节选）

```markdown
# 合同风险条款检查清单

## 通用检查清单

### 合同主体
- [ ] 名称与公章一致
- [ ] 地址、联系人、联系方式完整
- [ ] 法定代表人或授权代表签字完整

### 合同标的
- [ ] 标的名称、规格、型号明确
- [ ] 数量、质量标准明确
- [ ] 交付/履行地点、时间明确

### 对价条款
- [ ] 金额/报酬明确
- [ ] 支付方式、时间明确
- [ ] 发票类型、税率明确

### 违约责任
- [ ] 违约情形定义清晰
- [ ] 违约责任对等（双方义务相当）
- [ ] 违约金计算方式明确、合理

### 争议解决
- [ ] 管辖法院/仲裁机构明确
- [ ] 管辖地对公司有利（如约定在公司所在地）

### 知识产权
- [ ] 知识产权归属明确
- [ ] 使用许可范围明确
- [ ] 侵权责任承担明确

### 保密义务
- [ ] 保密信息定义明确
- [ ] 保密期限合理（一般不超过合同结束后2年）
- [ ] 违约责任明确

## 高风险条款特征

| 条款 | 高风险特征 | 建议修改 |
|------|-----------|---------|
| 违约责任 | 单方违约责任过重 | 调整为双方对等 |
| 解除条款 | 单方有权随时解除 | 增加提前通知期限和正当理由要求 |
| 免责条款 | 免责范围过于宽泛 | 限定为"不可抗力"等法定免责情形 |
| 知识产权 | 乙方所有知识产权归甲方 | 限定为"与本合同标的直接相关的" |
| 保密义务 | 永久保密义务 | 调整为"合同结束后2年内" |
```
